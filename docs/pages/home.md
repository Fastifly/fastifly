# Fastifly

Fastifly is a personal finance app you can run yourself.

It is built for people who want to track money clearly, share access carefully, and keep control of their own financial data.

## Why It Exists

Your finance app should not guess with money.

It should not create duplicate records because you clicked twice.

It should not make sharing an all-or-nothing choice.

Fastifly is being built around those basics first.

## What You Can Expect

Fastifly is designed to help you:

- keep your money records precise
- sign in safely
- recover access if something goes wrong
- invite another person without giving away full control
- run the app on your own server
- retry failed saves without accidentally creating duplicates

## Keep Your Data Close

Fastifly is self-hosted.

For a simple setup, it can run with one app and one SQLite database file. For larger setups, it can use PostgreSQL.

You should know where your data is stored and how it is backed up.

## Built Carefully Before Adding More

Fastifly is starting with the parts that protect your data:

- account access
- workspaces
- roles and permissions
- reliable database changes
- safe saves and retries

Reports, imports, budgets, rules, and mobile/offline features will be stronger because they will build on this foundation.

## The Promise

Fastifly should feel simple to use, but careful underneath.

Your financial history deserves software that treats every save as important.
