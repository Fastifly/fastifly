# Workspaces And Permissions

Fastifly lets finance data be shared without giving everyone the same level of control.

This matters for couples, families, partners, and anyone who manages money with another person.

## Start With Your Own Workspace

When a user creates an account, Fastifly creates a default workspace and ledger.

That gives the user a private place to start tracking money.

## Invite Someone Else

Users with the right permission can create invite links.

An invite link can give another person access to the workspace with a specific role.

## Roles Are Clear

Fastifly uses four workspace roles:

- owner
- admin
- editor
- viewer

Owners have full control. Admins can manage workspace access. Editors can work with finance data. Viewers can read without making changes.

## Access Can Be Changed

Workspace access is not permanent by default.

The right person can:

- invite someone
- revoke an unused invite
- accept an invite
- decline an invite
- list workspace members
- change a member role
- remove a member

## Owners Are Protected

Owner access is treated carefully.

A normal member edit should not accidentally remove or downgrade the workspace owner.

## What This Means

Sharing finance data should not mean losing control.

Fastifly is designed so each person has only the access they need.
